# Pritam Thapa — Portfolio

Personal portfolio and blog. Built with [Astro](https://astro.build).

```bash
npm install
npm run dev      # http://localhost:4321
npm run build    # → dist/
```

- **Add a blog post:** drop an `.mdx` file in `src/content/blog/`.
- **Edit site content:** `src/content/*.ts`.

Node 20+.
